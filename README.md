# Project Euler [![CircleCI](https://circleci.com/gh/n6g7/euler.svg?style=svg)](https://circleci.com/gh/n6g7/euler)

Solutions to [Project Euler](https://projecteuler.net) problems.

## How to?

  - install? `pip install -r requirements.txt`
  - run? `./run.py <n>`
  - test? `green`
